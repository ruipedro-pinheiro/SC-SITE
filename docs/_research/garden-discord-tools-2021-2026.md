# Garden [GRDN] Discord — community tool dump (2021 → 2026)

**Provenance**: pasted by Pedro on 2026-04-07 from the Garden [GRDN] org's Discord "tools/resources" channel. Five years of community-sourced Star Citizen tool recommendations. Treat as **authoritative crowd signal**, not marketing — these are tools that active SC veterans actually use and recommend to newcomers.

**Status**: reference material. The `sources-gap-fill` agent's job is to extract every tool/URL from this dump, dedupe against `SOURCES.md` and `seed-sources.md`, verify the NEW ones are alive, and add them to `SOURCES.md` with the same rigour as the other sources.

**Special attention** to these tools that are clearly NEW vs the team's existing catalog:

- **maps.adi.sc** — interactive 3D deck layouts for capital ships (Idris, Perseus, Hammerhead, Reclaimer, Polaris, Starfarer). Holoview layers showing internal components, fuses (with connections), medbeds, turrets. **This is a direct inspiration / potential data source for the 3D hangar mockup's "hardpoint markers" extension.**
- **deltaconsultingsc.com** — EFB-style station pad charts with pad locations per space station, sunrise/sunset times. Nobody else has this.
- **armory.thespacecoder.space** — FPS Gear Database (same author as Hauler already in seed-sources.md). Possible source for FPS equipment coverage.
- **rockbreaker.peacefroggaming.com** — mining rock-break calculator (mass, resistance, equipment config).
- **sc-cargo.space** — another cargo grid tool (author **bjax** is contactable in the Garden discord).
- **battlestations.osiris-devworks.com** — ship role/station assignment tool. Author intends to open an API once a milestone is hit.
- **SC 4.7 Mineable Resource Signal spreadsheet** — Google Sheet with mining signatures by resource, cross-referenced for quick scanning. Direct link (Pedro): https://docs.google.com/spreadsheets/d/1n4qqOfwbtsOubUTMWJ532pBGoFbx567S/edit?gid=943032706#gid=943032706
- **CStone Halo Belt route planner** — `cstone.space/resources/knowledge-base/36`, separate from the Item Finder Pedro already uses.
- **Cargo Grid Reference Guide** — RSI community hub post with isometric cargo grid PNGs for every flyable/drivable vehicle.
- **Buzz Killer HOTAS/HOSAS files** — joystick binding packs (Dropbox + Twitter). **Out of scope** for the data DB but note existence.
